# WhatsApp GPT Bot (Railway üçün)
Bu bot WhatsApp mesajlarına GPT-4o ilə avtomatik cavab verir.

## Quraşdırma

1. `.env` faylı yarat və OpenAI API açarını yerləşdir:
```
OPENAI_API_KEY=sk-...
```

2. Railway-də deploy et → QR kod çıxacaq → WhatsApp ilə skan et

## Başlamaq üçün

```bash
npm install
node index.js
```
